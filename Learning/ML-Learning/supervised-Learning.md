# 🎓 Supervised Learning

# 📖 What is Supervised Learning?

Supervised Learning is a type of Machine Learning in which the model learns from **labeled data**. Each input has a corresponding correct output (label). During training, the algorithm learns the relationship between the input features and the output labels. Once trained, the model can predict outputs for new, unseen data.

The word **"Supervised"** means that the learning process happens under the guidance of correct answers, just like a student learning from a teacher.

---

# 🔑 Key Points

- Uses labeled data.
- Every input has a corresponding output.
- Learns patterns from historical data.
- Used for prediction and classification.
- Improves performance with high-quality data.
- Most widely used Machine Learning technique.

# ⚙️ How Supervised Learning Works

### Step 1 — Collect Data

Collect labeled data from different sources.

### Step 2 — Data Preprocessing

Clean the dataset and remove missing values.

### Step 3 — Split Dataset

Split the data into

- Training Data
- Testing Data
 Predictions
### Step 4 — Train Model

The model learns patterns from the training dataset.

### Step 5 — Test Model

Evaluate the model using testing data.

### Step 6 — Make Predictions

The trained model predicts outputs for new data.

# 🧠 Common Algorithms
## 📈 Linear Regression

![Linear Regression](https://media.geeksforgeeks.org/wp-content/uploads/20260112155359063476/observed_value.webp)

### What is Linear Regression?

Linear Regression is a supervised learning algorithm used to predict continuous numerical values. It finds the best-fit straight line that represents the relationship between the input and output.

### Used For

- House Price Prediction
- Salary Prediction
- Sales Prediction

### Advantages

- Simple
- Fast
- Easy to understand

### Disadvantages

- Works only for linear relationships.
- Sensitive to outliers.

## 📉 Logistic Regression

![Logistic Regression](https://miro.medium.com/1*cJpAE_emjoQl4Ug-M4EXTw.jpeg)

### What is Logistic Regression?

Logistic Regression is a classification algorithm that predicts probabilities and class labels.

### Used For

- Spam Detection
- Disease Prediction
- Customer Churn

### Advantages

- Easy to implement
- High accuracy for binary classification

### Disadvantages

- Not suitable for complex relationships.

## 🌳 Decision Tree

![Decision Tree](https://www.xoriant.com/sites/default/files/uploads/2017/08/Decision-Trees-modified-1.png)

### What is Decision Tree?

Decision Tree divides data into branches based on conditions until a final decision is reached.

### Used For

- Loan Approval
- Disease Prediction
- Customer Segmentation

### Advantages

- Easy to understand
- Easy to visualize

### Disadvantages

- Can overfit.

## 🌲 Random Forest

![Random Forest](https://media.geeksforgeeks.org/wp-content/uploads/20251216121929631349/random_forest_algorithm.webp)

### What is Random Forest?

Random Forest combines multiple Decision Trees to improve prediction accuracy.

### Used For

- Fraud Detection
- Recommendation Systems
- Medical Diagnosis

### Advantages

- High Accuracy
- Reduces Overfitting

### Disadvantages

- Slow training.

## 📏 Support Vector Machine (SVM)

![SVM](https://media.geeksforgeeks.org/wp-content/uploads/20250807122724918613/linear_vs_non_linear_svm.webp)

### What is SVM?

Support Vector Machine finds the best boundary (hyperplane) to separate different classes.

### Used For

- Face Recognition
- Spam Detection
- Image Classification

### Advantages

- High Accuracy
- Works well with high-dimensional data

### Disadvantages

- Slow on very large datasets.

## 👥 K-Nearest Neighbors (KNN)

![KNN](https://media.geeksforgeeks.org/wp-content/uploads/20260120133056734500/k_nearest_neighbors.webp)

### What is KNN?

KNN classifies data based on the majority class of its nearest neighbors.

### Used For

- Pattern Recognition
- Recommendation Systems

### Advantages

- Easy to implement
- No training required

### Disadvantages

- Slow prediction for large datasets.

## 📊 Naive Bayes

![Naive Bayes](https://media.geeksforgeeks.org/wp-content/uploads/20250825100810938370/original_data.webp)

### What is Naive Bayes?

Naive Bayes is a probabilistic algorithm based on Bayes' Theorem.

### Used For

- Spam Detection
- Sentiment Analysis
- Text Classification

### Advantages

- Fast
- Good for text data

### Disadvantages

- Assumes all features are independent.
